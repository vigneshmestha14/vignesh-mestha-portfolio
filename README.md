# Vignesh Mestha — Data Engineer Portfolio

A modern, responsive portfolio website built with plain **HTML, CSS, and JavaScript** — no build step, fast, and free to host on GitHub Pages.

## Sections
- Hero / stats
- About
- Skills (Programming, Data Engineering, Azure, Modeling, Tools, Databases)
- Experience (LTM / LTIMindtree)
- Projects (Microsoft Fabric Medallion + CI/CD, Walmart DBT pipeline, MDIF automation, Flask apps)
- Education & Certifications (DP-600 / DP-203 / DP-900)
- Contact

## Local preview
Open `index.html` directly, or serve it:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```

## Deploy to GitHub Pages
1. Push this folder to a GitHub repo.
2. Repo **Settings → Pages → Build and deployment → Source: Deploy from a branch → Branch: `main` → `/ (root)`**.
3. Visit `https://<username>.github.io/<repo>/`.

The `resume.pdf` is linked via the "Resume ↓" button and downloads directly.
